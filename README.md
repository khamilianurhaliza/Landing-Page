
# Landing Page — Desain Figma

Desain landing page ini dibuat di Figma. Gambar dan aset visual sudah disimpan di folder assets dalam repositori ini.

**Figma:**

- Link desain: https://www.figma.com/design/Yeayrdkr0P4N53LNYM17Zw/Landing-Page?node-id=0-1&t=5zxXbXq4UgNjCEET-1

**Preview:**

- Buka folder [assets](assets) untuk melihat gambar dan ilustrasi yang digunakan pada desain.

**Cara menggunakan README ini:**

- **Link Figma:** buka link Figma di atas untuk melihat artboard, layout, dan spesifikasi desain (spacing, warna, font).
- **Aset:** gunakan file di folder [assets](assets) saat mengimplementasikan halaman. Simpan versi yang dioptimalkan untuk web (`.webp`/`.jpg`/`.png`) sesuai kebutuhan.
- **Export dari Figma:** jika perlu mengekspor aset baru, gunakan Figma > Export > pilih format dan ukuran yang sesuai (1x/2x untuk retina).

**Struktur proyek (singkat):**

- assets/ : semua gambar, ikon, dan ilustrasi desain.
- README.md : (file ini) dokumentasi singkat dan link Figma.

**Panduan singkat implementasi:**

- Ikuti grid dan spacing di Figma untuk menjaga konsistensi tampilan.
- Gunakan sistem font dan warna sesuai spesifikasi di Figma (lihat panel right-side di Figma untuk token warna dan style).
- Optimalkan gambar untuk web dan lazy-load untuk performa.

**Kontak / Kredit**

- Pembuat desain: lihat file Figma untuk metadata penulis.
- Jika perlu bantuan mengonversi desain ke HTML/CSS, beri tahu saya dan saya bantu membuat template dasar.
