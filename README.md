
# Landing Page

![Figma Preview](assets/image.png)

Ringkasan singkat proyek landing page ini. Ini adalah halaman statis modern yang dirancang untuk menampilkan produk atau layanan dengan tata letak bersih, tipografi yang jelas, dan ilustrasi yang menarik.

Badges: ![HTML](https://img.shields.io/badge/HTML-%23E34F26.svg?style=flat&logo=html5&logoColor=white) ![CSS](https://img.shields.io/badge/CSS-%231572B6.svg?style=flat&logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-%23323330.svg?style=flat&logo=javascript&logoColor=%23F7DF1E)

## Tangkasan
- **Nama proyek:** Landing Page
- **Deskripsi:** Template landing page responsif untuk pemasaran produk/layanan.
- **Lokasi file utama:** `index.html`, `assets/` (gambar dan aset)

## Fitur Utama
- Desain responsif untuk desktop, tablet, dan mobile
- Seksi Hero dengan CTA yang jelas
- Section fitur, paket harga, testimoni, dan footer
- Mudah dikustomisasi (warna, font, konten)

## Pratinjau Figma
Jika Anda memiliki hasil desain dari Figma, ekspor gambarnya (PNG/JPEG) dan letakkan di `assets/` dengan nama `image.png` atau `figma.png`. Gambar akan otomatis ditampilkan di bagian atas README.

Contoh langkah cepat:
1. Buka file Figma → pilih frame/artboard yang ingin dipratinjau
2. Klik kanan → Export → PNG (2x jika perlu)
3. Simpan ke `assets/image.png`

## Cara Menjalankan
1. Buka `index.html` di browser Anda (double click atau `Open with Live Server` jika menggunakan VS Code)
2. Untuk pengembangan cepat, gunakan ekstensi Live Server di VS Code atau jalankan server statis sederhana:

```bash
# menggunakan Python 3
python -m http.server 8000
# lalu buka http://localhost:8000
```
